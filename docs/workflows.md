# Workflow Documentation

## Daily Plan Workflow
Inputs:
- user_id

Steps:
1. Fetch habits
2. Fetch goals
3. Fetch mood/stress
4. Generate plan
5. Save to database

## Weekly Summary Workflow
Steps:
1. Fetch last 7 days of logs
2. Analyze habits
3. Analyze mood
4. Detect patterns
5. Generate summary

## AI Coach Workflow
Steps:
1. Gather user context
2. Build conversation memory
3. Generate assistant response
